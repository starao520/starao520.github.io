centos7 安装node

2019-10-15

1、安装wget，如果之前没有安装的话

​ yum install -y wget

2、去官网查看最新的长期维护版本

​ wget https://nodejs.org/dist/v10.13.0/node-v10.13.0-linux-x64.tar.xz

3、解压下载的node包

​ xz -d node-v10.13.0-linux-x64.tar.xz

​ tar -xf node-v10.13.0-linux-x64.tar

4、配置node与npm。假设当前node目录为/root/node-v10.13.0-linux-x64

​ ln -s /root/node-v10.13.0-linux-x64/node /usr/bin/node

​ ln -s /root/node-v10.13.0-linux-x64/npm /usr/bin/npm

5、查看node与npm版本，如果成功就完成了

​ node -v

​ npm -v

6、更新node与node

​ npm install -g npm 更新npm

​ npm cache clean -f 清除npm缓存

​ npm install -g n 安装n模块

​ ln -s /root/node-v10.13.0-linux-x64/bin/n /usr/bin/n 配置n模块

​ n stable 升级node.js到最新稳定版本

node npm

centos7 安装nginx步骤

2019-10-15
1、安装所需依赖

​ yum -y install gcc zlib zlib-devel pcre-devel openssl openssl-devel

2、下载并解压安装包

​ wget http://nginx.org/download/nginx-1.13.7.tar.gz

​ tar -xvf nginx-1.13.7.tar.gz

3、安装nginx

​ cd nginx-1.13.7

​ ./configure

​ make

​ make install

4、启动nginx

​ /usr/local/nginx/sbin/nginx

5、设置开机启动

​ 1.在/etc/init.d下创建文件nginx

​ vi /etc/init.d/nginx

​ 2.输入官方文档内容

​ 3.修改其中配置为

​ nginx=”/usr/local/nginx/sbin/nginx”

​ NGINX_CONF_FILE=”/usr/local/nginx/conf/nginx.conf”

​ 4.保存后设置文件执行权限

​ chmod a+x /etc/init.d/nginx

​ 5.将nginx服务添加到chkconfig管理列表中

​ chkconfig --add /etc/init.d/nginx

​ 6.设置开机启动

​ chkconfig nginx on

docker打包镜像

2019-10-11

操作环境为centOS7

1、下载代码到指定目录

​ get clone xxx.git

2、进入到项目根目录

​ cd /xxx

3、将打包好的服务（server.jar）放在根目录下

4、创建Dockerfile文件

​ vim Dockerfile

​ #获取对应jdk

​ FROM yuanziqiang1995/openjdk-maven:3.0

​ #添加打包好的jar包，并重命名

​ ADD server.jar ams-center.jar

​ #设置对应的端口号，与服务的yml文件相对应

​ EXPOSE 8080

​ #生成docker入口

​ ENTRYPOINT ["java", "-Djava.security.egd=file:/dev/./urandom", "-jar", "ams- center.jar"]

5、修改项目的pom.xml文件

6、对项目进行编译

​ docker build -t ams-center .#后面有个.

7、执行打包的docker

​ docker run -p 8080:8080 --name ams-center ams-center:latest

8、查看是否成功

​ docker ps
