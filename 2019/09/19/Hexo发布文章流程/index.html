Hexo发布文章流程

2019-09-19

1、进入Hexo根目录
cd wrokDoc

2、添加文章
hexo new '文章标题'

3、编辑文章
进入到workDoc/source/_post目录下，编辑新添加的文章.md

4、编译文章
hexo g

5、本地查看文章
hexo s
通过localhost:4000进行本地查看

6、清空本地缓存
hexo clean

7、提交到github
hexo g -d
