centOS7通过Nginx配置https访问

2019-04-23

1.安装openssl 和 openssl-devel
yum -y install openssl openssl-devel

2.切换到nginx的源码包安装位置
cd /usr/local/nginx/nginx/nginx-1.8.0

3.执行语句,安装ssl模块
./configure –prefix=/usr/local/nginx –with-http_stub_status_module –with-http_ssl_module

4.重新编译配置
make

5.备份旧的nginx
cp /usr/local/nginx/sbin/nginx /usr/local/nginx/sbin/nginx.bak

6.关闭nginx
ps -ef|grep nginx
kill -QUIT 13542

7.替换nginx
cp ./objs/nginx /usr/local/nginx/sbin/

8.重启nginx
cd /usr/local/nginx/sbin/
./nginx -s reload

9.将证书中解压出来的两个文件(.crt文件和.key文件)复制到nginx的目录中/usr/local/nginx/conf中

10.配置http和https同时访问

httpscentos7nginx
