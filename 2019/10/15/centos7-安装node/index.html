centos7 安装node

2019-10-15

1、安装wget，如果之前没有安装的话

​ yum install -y wget

2、去官网查看最新的长期维护版本

​ wget https://nodejs.org/dist/v10.13.0/node-v10.13.0-linux-x64.tar.xz

3、解压下载的node包

​ xz -d node-v10.13.0-linux-x64.tar.xz

​ tar -xf node-v10.13.0-linux-x64.tar

4、配置node与npm。假设当前node目录为/root/node-v10.13.0-linux-x64

​ ln -s /root/node-v10.13.0-linux-x64/node /usr/bin/node

​ ln -s /root/node-v10.13.0-linux-x64/npm /usr/bin/npm

5、查看node与npm版本，如果成功就完成了

​ node -v

​ npm -v

6、更新node与node

​ npm install -g npm 更新npm

​ npm cache clean -f 清除npm缓存

​ npm install -g n 安装n模块

​ ln -s /root/node-v10.13.0-linux-x64/bin/n /usr/bin/n 配置n模块

​ n stable 升级node.js到最新稳定版本
