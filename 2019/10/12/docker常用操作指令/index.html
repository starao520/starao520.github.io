linux常用操作指令

2019-10-12

1、查看本机所有镜像

​ docker images

2、查看本机docker运行情况

​ dcoker ps -a -q

3、通过通配符查找镜像

​ docker images ce* #查找包含ce的镜像

4、查看镜像详情

​ docker inspect name #name为镜像名称

5、搜索镜像

​ docker search tomcat #搜索包含tomcat的镜像

6、删除镜像

​ docker rmi name #name为镜像名称，删除多个用空格隔开
