windows搭建Hexo服务

2019-09-19

1、安装Node.js 和 git

2、注册github

3、安装Hexo
$ npm install -g hexo-cli

4、新建目录
hexo init <folder>
cd <folder>
npm install
npm install hexo-deployer-git --save

5、配置_config.yml文件
修改最后一行的配置：
deploy
type: git
repository: https://github.com/yourname/yourname.github.io
branch: master
